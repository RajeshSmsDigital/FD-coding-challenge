# FD-coding-challenge
Coding challenge for FD

# Steps to run this project
first download or clone the repository

# Step 1: Add required things

- latest Node version of v16
- Typescript

## Step 2:  Run Frontend

- #`cd frontend`
- #`npm install`
- #`npm start`

## Step 3:  Run Backend
- #`cd backend`
- #`npm install`
- #`tsc`
- #`node .`
